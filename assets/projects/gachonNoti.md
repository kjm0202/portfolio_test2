# 가천 알림이 (Gachon Noti)

## 프로젝트 개요
가천대학교 학생들을 위한 알림 서비스입니다.

## 주요 기능
- 📢 학교 공지사항 알림
- 📅 학사 일정 알림
- 🍽️ 오늘의 학식 메뉴
- 📚 도서관 좌석 현황

## 기술 스택
- **Framework**: Flutter Web
- **Backend**: Firebase Functions
- **Scraping**: Puppeteer
- **Notification**: FCM

## 특징
- PWA 지원
- 카테고리별 알림 설정
- 키워드 알림 기능

## 개발 기간
2022년 9월 ~ 2023년 2월
