# CONNECT

## 프로젝트 개요
사용자 간 연결을 위한 소셜 네트워킹 플랫폼입니다.

## 주요 기능
- 👥 사용자 프로필 관리
- 💬 실시간 채팅
- 📝 피드 및 게시물
- 🔍 사용자 검색 및 추천

## 기술 스택
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Real-time**: Socket.io
- **Authentication**: JWT

## 아키텍처
- RESTful API 설계
- Microservice 구조
- Redis 캐싱

## 개발 기간
