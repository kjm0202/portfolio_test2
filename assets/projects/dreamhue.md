# DreamHue

## 프로젝트 개요
수면에 불편을 겪는 사람들을 위한 사운드 앱입니다.

## 주요 기능
- 🌙 다양한 자연 사운드 제공
- 🎵 사용자 맞춤형 사운드 믹싱
- ⏰ 수면 타이머 기능
- 📊 수면 패턴 분석

## 기술 스택
- **Framework**: Flutter
- **State Management**: Provider
- **Audio**: just_audio
- **Storage**: Hive
