# Portfolio Website

## 프로젝트 개요
Flutter Web으로 구축된 반응형 포트폴리오 웹사이트입니다.

## 주요 기능
- 🎨 다크/라이트 테마 지원
- 🌐 다국어 지원 (한국어/영어)
- 📱 반응형 디자인
- ✨ 보케 배경 애니메이션

## 기술 스택
- **Framework**: Flutter Web
- **Localization**: flutter_localizations
- **Animation**: Custom Canvas Painter

## 특징
- 모바일, 태블릿, 데스크톱 모두 지원
- 시스템 테마 자동 감지
- 부드러운 스크롤 애니메이션
